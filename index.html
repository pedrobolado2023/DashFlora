📋 Fluxo de Processamento

  1. Webhook recebe mensagem do WhatsApp via Evolution API
  2. Switch detecta tipo: texto ou áudio
  3. Consulta se lead existe no Supabase (tabela dados_cliente)
  4. Cria lead se não existir com telefone e nome WhatsApp
  5. Envia presença "digitando" no WhatsApp
  6. Reage à mensagem com emoji
  7. Marca como lida a mensagem
  8. Armazena no Redis temporariamente (15s)
  9. Aguarda batching de mensagens
  10. Consolida histórico e envia para AI Agent
  11. AI Agent processa com memória do PostgreSQL
  12. Aciona agentes especializados conforme necessário
  13. Formata resposta para WhatsApp
  14. Envia resposta final via Evolution API
